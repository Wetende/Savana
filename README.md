# Savana

Savana is a premium coffee project with a dedicated frontend in `frontend/` and a backend built with Django and MySQL.

## Project Structure

- `frontend/`: React + Vite + JavaScript 
- `backend/`: Django + MySQL application

## Tech Stack

- Frontend: React + Vite + JavaScript
- Backend: Django
- Database: MySQL

L.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```
