# Savana

Savana is a premium coffee project with a dedicated frontend in `frontend/` and a backend built with Django and MySQL.

## Project Structure

- `frontend/`: React + Vite + JavaScript landing page
- `backend/`: Django + MySQL application

## Tech Stack

- Frontend: React + Vite + JavaScript
- Backend: Django
- Database: MySQL

## Frontend

The current frontend focuses on a premium wholesale coffee landing experience, while the backend is structured around Django and MySQL.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```
